!Question--1.
What if i want to replace string from end , at middle and from a specific position .
!Ans--1
1. Replace from the end (right side)

Use rsplit() + join()

Example: Replace from end
text = "one two two two"

# Replace last "two" with "three"
parts = text.rsplit("two", 1)
text = "three".join(parts)

print(text)

rsplit -- reverse split -- breaks string from right to left and the second argumnet (integer) will tell, from where to split string like in this code it will be like ['one two two' , 'two'].
join() connects list into a string using "three" in between.

$ Q---a. so instead of join i can't use append hmm ? what is the difference.
$ ans--a. append() → adds item to end of list
join() → joins list into string by putting a string between list elements


-------------------------------------------------------------------


2. Replace from the middle

Use slicing + replace

Example:

Replace only in middle part:

text = "apple banana mango banana apple"

# Suppose we want to replace only middle banana
middle = text[6:24]          # take middle part
middle = middle.replace("banana", "orange", 1)

text = text[:6] + middle + text[24:]

print(text)


-------------------------------------------------------------------


3. Replace at a specific position (best method)

Use find() and string slicing

Example:
text = "one two two two"

index = text.rfind("two")   # find from right
text = text[:index] + "three" + text[index+3:]

print(text)

ans---
find method is used to find index of the item. we pass the item as an argument insidde parenthesis and it return it index and if not found then it will return -1.
There is an another which is used to find positions of items which is index method.(index()).
